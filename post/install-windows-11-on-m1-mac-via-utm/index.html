使用UTM安装Win 11 ARM

2023-01-13 2 min read # 技术

在Apple Silicon Mac上安装Windows 11

下载慢的请看利用Doget快速下载GitHub资源
1.下载UTM
2.打开DMG文件，并将其拖拽到Application文件夹

3.下载ARM版Win 11镜像：

  • ed2k
  • magnet
    4.打开UTM，选择新建一个虚拟机

    在打开的菜单中选择虚拟化

    选择Windows

    将Install Windows 10 or higher和Install drivers and spice guest tools选中，在启动ISO中选中前面下载的镜像

    内存建议选择8192MB，CPU核心默认

    存储建议默认（足够用了）

    共享目录可以稍后设置

    点击完成结束虚拟机的创建

    P.S. 如果你的硬盘空间不够，可以点击这个按钮将VM移到其他地方
    5.开启虚拟机（切记：在出现下列情况时，立马按下键盘上的任意键）

    6.按照正常流程安装Windows
    7.遇到下面情况时，按下fn+shift+F10打开命令提示符，输入regedit打开注册表编辑器


    在HKEY_LOCAL_MACHINE/SYSTEM/Setup下新建一个名字为LabConfig的项
    在这个项中建立名为BypassTPMCheck的DWORD值并将其值设为1
    再建立名为BypassSecureBootCheck的DWORD值并也将其设置为1

    8.退出regedit和cmd，返回安装程序的上层菜单，再点击下一步，继续安装
    9.在这个界面中选择Custom：

    10.继续安装，等待安装完毕

    11.安装完毕后关闭虚拟机，弹出ISO镜像进行Win11初始设置。
    12.在这个界面中，按fn+shift+F10打开命令提示符，

    输入
    oobe\bypassnro
    13.点击这个按钮选择Install Windows Guest Tools...
    14.进入Windows系统，进入CD驱动器，安装Spice Guest Tools（一路下一步）
    结束安装后选择I want to manually reboot later.
    15.关闭虚拟机，打开虚拟机设置，在显示中勾选高分辨率模式

安装完成，享受你的Windows吧😀

如果spice guest tools未自动安装，请从这里下载iso文件并通过上述方法挂载到Windows中。
